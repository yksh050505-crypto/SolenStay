# SolenStay

1·2·3호점 예약·청소 관리 앱.

- **모바일**: Flutter (Android native APK + Web/PWA)
- **백엔드**: Firebase (Auth · Firestore · Functions · Storage · Hosting · FCM)
- **자동화**: Zapier (에어비앤비 iCal → Google Calendar → 앱)
